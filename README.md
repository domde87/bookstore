# Litterae - Curated Books

A beautifully designed bookstore web application featuring a curated collection of books with an elegant, refined aesthetic.

## Features

- **Elegant Design**: Premium typography with Cormorant Garamond and Inter fonts
- **Responsive Layout**: Optimized for all device sizes
- **Book Collection**: Browse and discover curated books
- **Modern Color Palette**: Sophisticated cream, sage, and gold color scheme
- **User-Friendly Interface**: Intuitive navigation and smooth interactions

## Design Palette

- **Cream**: #faf8f3
- **Dark Brown**: #3a3028
- **Sage**: #7a8f6b
- **Gold**: #c9b896
- **Warm Beige**: #e8dcc8

## Getting Started

Simply open `index.html` in your web browser to view the application.
